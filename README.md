# 🚀 Currency Converter App

Una aplicación web moderna y minimalista para la conversión de divisas
en tiempo real. Este proyecto forma parte de mi stack de desarrollo como
**Software Engineer**, enfocado en crear interfaces limpias y funcionales.

## 🚀 Demo en Vivo

Puedes ver el proyecto funcionando aquí:  
[Ver Prueba Módulo JavaScript](https://nachorob.github.io/prueba-modulo-javascript/)

## ✨ Características

-   **Conversión en Tiempo Real:** Conexión con APIs externas para
    obtener tipos de cambio actualizados.
-   **UI Moderna:** Diseño basado en **Inter** y **JetBrains Mono**,
    optimizado para macOS.
-   **Responsive Design:** Adaptable a dispositivos móviles y desktops.
-   **Micro-interacciones:** Efectos de elevación y transiciones suaves
    en botones e inputs.

## 🛠️ Stack Tecnológico

-   **Frontend:** HTML5, CSS3 (Flexbox & Grid).
-   **Lógica:** JavaScript (Vanilla).
-   **Herramientas:** Desarrollado en **VS Code** desde un **MacBook Pro
    M4 Pro**.

## 🚀 Instalación y Uso

1.  Clona el repositorio:

``` bash
git clone https://github.com/NachoRob/currency-converter-app.git
```

2.  Abre el archivo `index.html` en tu navegador o usa la extensión
    **Live Server** en VS Code.

## 📂 Estructura del Proyecto

    .
    ├── index.html       # Estructura principal
    ├── style.css        # Estilos (Midnight & Azure theme)
    ├── script.js        # Lógica de consumo de API
    └── .gitignore       # Configuración para macOS y entornos locales
